# Strudel Visuals (Experimental) --- English above

**Strudel Visuals** é um experimento que explora a criação de uma camada gráfica baseada em **HTML5 Canvas** integrada ao **Strudel REPL**.

A ideia é oferecer uma alternativa leve ao Hydra para experimentações visuais diretamente a partir dos padrões musicais do Strudel, utilizando JavaScript puro e Canvas 2D.

> **Status:** Experimental. Esta biblioteca ainda está em desenvolvimento e sua API pode mudar significativamente nas próximas versões.

## Objetivos

O projeto busca investigar uma API visual inspirada na filosofia do Strudel, onde padrões musicais possam controlar diretamente elementos gráficos.

Algumas ideias em desenvolvimento incluem:

* grids e layouts reativos;
* formas geométricas;
* padrões visuais sincronizados com padrões musicais;
* parâmetros visuais controlados por Patterns;
* integração com recursos nativos do Strudel.

## Estado atual

Atualmente a biblioteca implementa:

* um motor gráfico baseado em HTML5 Canvas;
* múltiplas camadas visuais;
* integração com `register()` através de `.visuals()`;
* grids parametrizáveis;
* controle das células por Patterns do Strudel.

Trata-se de um protótipo de pesquisa. O código prioriza experimentação e exploração de ideias, não estabilidade ou desempenho.

# Strudel Visuals (Experimental)

**Strudel Visuals** is an experimental project that explores a **HTML5 Canvas** visual layer integrated with the **Strudel REPL**.

The goal is to provide a lightweight alternative to Hydra for visual experimentation directly from Strudel patterns using plain JavaScript and Canvas 2D.

> **Status:** Experimental. The API is still evolving and may change considerably between versions.

## Goals

This project investigates a visual API inspired by the philosophy of Strudel, allowing musical patterns to control graphical elements directly.

Current ideas include:

* reactive grids and layouts;
* geometric primitives;
* visual patterns synchronized with musical patterns;
* visual parameters driven by Patterns;
* deeper integration with native Strudel features.

## Current state

At the moment the library provides:

* an HTML5 Canvas rendering engine;
* multiple visual layers;
* integration with `register()` through `.visuals()`;
* configurable grids;
* Pattern-driven cell activation.

This is a research prototype focused on experimentation rather than stability or performance.
